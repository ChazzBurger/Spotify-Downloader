# Spotify Downloader
### *Make sure that your playlist is public or else it will not download!*

A program that downloads songs from spotify

- **Includes support for downloading:**
  - Single Tracks
  - Albums
  - Playlists
- **Includes all metadata within each song including:**
  - Title
  - Artists
  - Album
  - Album Image
  - Spotify, Youtube IDs
- **Normalization of audio to keep all songs the same audio level**
- **Small file sizes (~1MB per min of audio)**
